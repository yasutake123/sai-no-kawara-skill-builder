# 賽の河原スキルビルダー
## **1. アプリケーション概要**

- **一言コンセプト**：学んだスキルを石として積み上げ、放置すると鬼に崩される、賽の河原風・忘却防止学習アプリ。
- **ターゲット層**：プログラミング学習者（JavaやReactなどを勉強中の人）。
- **開発の目的**：
    - 各言語の「学習の定着度」を視覚的に楽しみ、忘却を防ぐ。
    - 復習のモチベーションをゲーム感覚（ゲミフィケーション）で維持する。

## **2. 主要機能一覧（フェーズ別）**

開発の優先順位を分けるためのリストです。

- **【最優先：フェーズ1】コアロジックと基本UI（Reactのみで完結）**
    - 言語の手動セレクト方式（Java, React, Python等のボタンクリックで石を積む）
    - デバッグ用「時間を進める（鬼を呼ぶ）」ボタン
    - 時間経過（ボタン操作）による石の耐久値（HP）減少システム
    - 復習ボタンによる石の耐久値回復機能
    - 復習回数に応じた「忘却耐性（HPの減りにくさ）」アップシステム
    - サイドバーへの詳細情報表示（忘却曲線グラフのモック表示）
- **【優先：フェーズ2】データの永続化とビジュアル強化**
    - ローカルストレージ（ブラウザ）へのデータ保存機能
    - 【新規】GitHub風の芝生（コミットグラフ風の学習履歴ヒートマップ）の搭載
    - 石が積まれる・崩れるアニメーション演出（Framer Motion等）
- **【将来：フェーズ3】完全自動化連携**
    - GitHub APIとの連携（プッシュされたコードを自動取得して解析・自動で石を積む）

## **3. 画面遷移・画面イメージ（3カラム構成）**

画面にどんな要素を置くかを文字で整理します。

- **メイン画面（1画面集中型・左右分割レイアウト）**
    - **左側（メインエリア）**：
        - 上部：言語選択ボタン（Javaを積む、Reactを積む等）
        - 中央：積み上げられた石のグラフィック表示エリア（下から上に積み上がる）
        - 下部：デバッグ用「時間を1日進める（鬼を呼ぶ）」ボタン
        - 最下部：GitHub風の芝生（学習履歴ヒートマップ）エリア
    - **右側（サイドバーエリア）**：
        - 任意の石をクリック（またはホバー）した際に開く。
        - 選択した石の「エビングハウスの忘却曲線グラフ」や「現在の復習回数（耐久度ボーナス）」を表示。
- **設定画面（フェーズ4以降）**
    - GitHubのアカウント連携やリポジトリ選択

## **4. ゲームルール・ロジック詳細**

このアプリの心臓部となる数値やルールの設定です。

- **石の種類と判定ルール**
    - 初期は言語単位（ `java` , `html/css`, `react` , `python` 等）に対応する「石コンポ―ネント」を生成。
- **鬼の襲来（風化）ルール**
    - 「時間を進める」ボタンを1回押すごとに、すべての石のHPが減少する（ベース減少値：1日につき20%）。
    - HPが0%になった石は消滅する（鬼に粉砕された状態）。
- **【新規】熟練度システム（復習による減りにくさ）**
    - 各石は `reviewCount（復習回数）` のデータを持つ。
    - 復習ボタンを押すたびにHPが100%に全回復し、`reviewCount` が `+1` される。
    - 2回目以降のダメージは、復習回数に応じて軽減される（例：ダメージ ＝ `20% / (reviewCount + 1)` ）。
- **救済・進化ルール**
    - 何度も修復して一定以上の復習回数（例：5回以上）に達した石は「マスターの岩」に進化し、鬼が来ても絶対に崩れなくなる。

## **5. 技術スタック（予定）**

- **フロントエンド**：React (Vite)
- **スタイル/アニメーション**：Tailwind CSS / Framer Motion
- **グラフ・ヒートマップ用ライブラリ**：
    - `Recharts`（サイドバーの忘却曲線グラフ用）
    - `react-calendar-heatmap`（GitHub風の芝生用）
- **その他**：GitHub API（将来用）

## **🔍 各エリアの詳しいUI設計**

## **1. 左サイドバー（追加ボタン ＆ 鬼召喚）**

- **言語追加エリア**：最初から並んでいる「Java」「React」などの横の`[+]`を押すと、中央に石が積まれます。最下部に`[+] 新しい言語` ボタンがあり、これを押すと自分で「PHP」や「TypeScript」など**新しいボタンを自由に追加**できます。
- **鬼を呼ぶ（デバッグ）ボタン**：時間で自動的に崩れるのではなく、**「自分のタイミングでいつでも鬼を呼べる（時間を進められる）」**仕様にします。これにより、「あと数時間で崩れちゃう！」というリアルタイムの焦りが完全に消え、自分のペースで「よし、そろそろ鬼を呼んで耐久テストをしてみよう」と気楽に構えることができます。

## **2. 中央エリア（賽の河原メイン）**

- **スクロールなし**：石が積み上がっていく様子が、画面中央にポン、ポンと縦に重なっていきます。背景は少しグラデーションのある穏やかな和風の川原のデザインです。
- **ホバー（カーソルを合わせる）**：石の上にマウスを乗せると、吹き出し（ポップアップ）で「Java / 耐久値60%」と、最低限の情報だけがふわっと浮き出ます。画面が文字でごちゃつかず、スッキリした見た目を維持できます。

## **3. 右サイドバー（芝生 ＆ 忘却曲線）**

- **初期状態**：何も選択していないときは、閉じて（非表示に）なっているか、もしくは「石を選択すると詳細が表示されます」というシンプルな和風のイラストが表示されています。
- **クリック時**：中央の石をクリックするとスッと開き、その石の「忘却曲線（グラフ）」と、このアプリの総まとめである**「GitHub風の芝生（カレンダーヒートマップ）」**が出現します。

## **💡 この設計の素晴らしいところ（UXのメリット）**

- **マイペースに楽しめる**：勝手にカウントダウンされないため、1週間アプリを開かなくても、自分のタイミングでログインして「復習」や「鬼の召喚」ができます。
- **コレクション感覚**：左サイドバーでボタンを増やし、中央に色々な色の石が積み上がっていく様子は、まるでアクアリウムや盆栽を育てるような「所有欲」と「癒やし」を満たしてくれます。
